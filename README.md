#Онлайн пианино
Coolpiano.ru – это виртуальное онлайн пианино, на котором можно играть с помощью клавиатуры. На пианино отображаются ноты и номера клавиш. Доступно два варианта раскладки. Играть музыку можно бесплатно и без регистрации. Диапазон фортепиано – 5 октав (61 клавиша). Ссылка: [Онлайн пианино](http://coolpiano.ru)
